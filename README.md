# ファンノート

好きな人の X・note・YouTube・Suno のページに、すぐ飛べる わたし用のランチャー。
きろく帖の姉妹アプリです。スマホ向けPWA（ホーム画面に追加できます）。

## いま入っているもの（Phase 1〜4）

- 人カード（アイコン・名前・SNSボタン）
- 上のフィルターチップ（すべて / note / X / YouTube / Suno）で絞り込み
- SNSボタンをタップで、その人のページに飛ぶ
- ＋おきに追加（新しい人 / もういる人に追加、置き換え確認つき）
- 詳細シート（名前・アイコンの編集、SNSを足す・外す、人を削除）
- **note の新着NEW**：起動時に最新記事を見に行き、前に見たときより新しければ note ボタンに「NEW」。タップして開くと既読になって消えます。右上の ⟳（大きめ）で手動更新も可。詳細シートに最新記事のタイトルも出ます。最新は「公開日がいちばん新しい記事」で判定（固定記事に引っぱられない）。
- **SNSボタン**：カードのボタンは **SNS名だけ**を表示（NEWバッジは残す）。ID（neko_art 等）は詳細シートで確認。
- **アイコン**：追加時は自動で既定（note＝本物の画像／X・YouTube＝絵文字）。強制の選択画面は出しません。変えたい時は詳細シートで（絵文字を選ぶ／note の人は画像⇄絵文字の切替も可）。
- **フォロー中から選ぶ**：おきに追加→note で、あなたの note ID（任意入力・一度だけ／端末に保存）からフォロー中を一覧表示してまとめて追加。
- シートは下スワイプ／✕／背景タップで閉じます（スワイプがアプリ本体に伝わって閉じないように対策）。
- **今日のチェック＆沈み**：SNSボタンを開くと「今日見た」＝✓。見終わった人は「今日チェック済み」の下に薄く沈みます。
- **今日◯/◯（フィルター連動）**：上のバーで進捗。すべて＝全員のうち全部見終わった人／単一SNS＝そのSNSを見た人。
- **夜中の自動リセット**：端末の日付が変わると、起動時・復帰時に今日の記録（✓・沈み）がリセット（全員が上に戻る）。NEWはリセットされません（未読は翌日もNEW）。
- **詳細シート**：「✓今日はこの人OK」／済みなら「↩まだに戻す」。
- **並べ替え**：カード右の ⠿ をつかんで上下にドラッグ → 順番を保存。
- **対応SNS**：note / X / YouTube / **Suno**（Sunoは手動入力・飛ぶのみ／自動取得・NEWなし）。レジストリに1行で増やせます。
- 保存はこの端末の中だけ（IndexedDB）。サンプルの人が最初から入っています。

## まだ入っていないもの（Phase 4 の残り・任意）

- SNS別「日次チェックに含めない」トグル、YouTube の NEW（RSS）など

## 技術メモ

- HTML / CSS / JavaScript だけ（サーバー不要）
- note 取得は note の公開APIを中継（きろく帖の Cloudflare Worker プロキシを流用）ごしに呼び出し：`{NOTE_PROXY}/?path=`（APIパスを encodeURIComponent）。切り替えは `js/app.js` の `NOTE_PROXY` で。
- 配色：ラベンダー `#9F8FD4` ／ 生成り `#FAF7F0`
- 公開：GitHub Pages

## アイコンの作り直し

`icons/make_icons.py` を実行すると、アイコン画像を作り直せます（Python + Pillow）。

```
cd icons && python3 make_icons.py
```
