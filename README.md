# ファンノート

好きな人の X・note・YouTube のページに、すぐ飛べる わたし用のランチャー。
きろく帖の姉妹アプリです。スマホ向けPWA（ホーム画面に追加できます）。

## いま入っているもの（Phase 1〜2）

- 人カード（アイコン・名前・SNSボタン）
- 上のフィルターチップ（すべて / note / X / YouTube）で絞り込み
- SNSボタンをタップで、その人のページに飛ぶ
- ＋おきに追加（新しい人 / もういる人に追加、置き換え確認つき）
- 詳細シート（名前・アイコンの編集、SNSを足す・外す、人を削除）
- **note の新着NEW**：起動時に最新記事を見に行き、前に見たときより新しければ note ボタンに「NEW」。タップして開くと既読になって消えます。右上の ⟳ で手動更新も可。詳細シートに最新記事のタイトルも出ます。
- 保存はこの端末の中だけ（IndexedDB）。サンプルの人が最初から入っています。

## まだ入っていないもの（Phase 3 以降）

- 「今日この人を見たか」の日次チェック・見終わった人が下に沈む・夜中のリセット・カウンター

## 技術メモ

- HTML / CSS / JavaScript だけ（サーバー不要）
- note 取得は note の公開APIを中継（きろく帖の Cloudflare Worker プロキシを流用）ごしに呼び出し：`{NOTE_PROXY}/?path=`（APIパスを encodeURIComponent）。切り替えは `js/app.js` の `NOTE_PROXY` で。
- 配色：ラベンダー `#9F8FD4` ／ 生成り `#FAF7F0`
- 公開：GitHub Pages

## アイコンの作り直し

`icons/make_icons.py` を実行すると、アイコン画像を作り直せます（Python + Pillow）。

```
cd icons && python3 make_icons.py
```
